OpenAI Ads Manager 업로드 워크북 작성

광고주 등록/정산 정보는 별도 트래커에서 관리하고, 이 페이지는 Ads Manager 벌크 업로드에 필요한 캠페인·광고그룹·광고 소재 정보를 수집합니다.

운영용 입력 페이지

재오픈 전 draft 기준

수동 세팅 기준과 벌크 업로드 기준을 분리해 확인합니다

수동 세팅 기준 필수 입력

  • 시작일, 종료일, 예산 유형, 캠페인 목표는 내부 검토와 Ads Manager 수동 세팅을 위해 모두 입력합니다.
  • Clicks(CPC)는 광고그룹별 max_bid를 KRW 기준으로 입력하고, Views(CPM)는 max_bid를 비웁니다.
  • 캠페인명, 광고그룹명, 소재명은 영문/숫자/공백/하이픈/언더스코어만 사용합니다.

벌크 업로드 기준 주의사항

  • 다운로드 파일은 campaigns / adgroups / ads 시트로 생성됩니다.
  • 한국 집행은 target_countries를 빈칸(NULL)으로 두는 것을 기본값으로 사용합니다.
  • 업로드 후 Ads Manager에서 국가, 스케줄, 입찰, 소재 URL 반영값을 항목별로 다시 확인합니다.

워크북 파일 불러오기

광고주가 작성한 공식 워크북(.xlsx)을 업로드하면 campaigns / adgroups / ads 값을 검사하고, 오류가 없을 때 아래 입력폼에 자동으로 채웁니다.
