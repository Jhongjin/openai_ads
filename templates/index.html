    OpenAI 직접 집행은 최대 100자이며 32~48자를 권장합니다. 크리테오 경유는 최대 60자이며 국문 띄어쓰기를 포함합니다.

  • 랜딩 URL OpenAI 공식 문서

    유효하고 접근 가능한 URL이어야 하며, 광고 내용과 관련성이 높은 페이지를 권장합니다. 홈 대신 제품·컬렉션·콘텐츠 페이지가 좋습니다. UTM 등 추적 파라미터는 대상 URL에 직접 추가할 수 있습니다.

랜딩 페이지 기술 요건 — 광고주 개발팀이 확인할 것

OpenAI 공식 문서|OpenAI 공식
  • OpenAI 광고 로봇 허용 OpenAI 공식 문서

    robots.txt에서 OAI-AdsBot은 반드시 허용해야 합니다. OAI-SearchBot도 허용을 권장합니다. 예: User-agent: OAI-AdsBot / Allow: /

  • 웹 방화벽/봇 완화 예외 처리 OpenAI 공식 문서

    Cloudflare·Akamai 등 웹 방화벽이나 봇 완화 설정에서 OpenAI 크롤러 user-agent를 허용 목록에 추가합니다. 단, OAI-AdsBot은 현재 Cloudflare에서 공식 검증·허용 목록 등록이 완료된 상태입니다.

  • CAPTCHA·JS 챌린지 예외 처리 OpenAI 공식 문서

    CAPTCHA, JS 챌린지, 사람 확인 로직이 있으면 OpenAI 크롤러가 랜딩 페이지를 확인할 수 있도록 예외 처리합니다.

  • 속도 제한 확인 OpenAI 공식 문서

    대량 업로드나 반복 검수 시 429(Too Many Requests) 등 속도 제한이 발생하지 않도록 서버 정책을 확인합니다.

  • 사전 자가점검 안내 OpenAI 공식 문서

    이 사이트의 [랜딩 URL 검사] 탭에서 robots.txt 기준 1차 자가점검이 가능합니다. 최종 확정은 광고주 개발팀의 실제 OpenAI 광고 로봇 접근 테스트가 필요합니다.

집행 조건 — 영업팀이 광고주에게 안내할 것

나스미디어 내부 자료|확정 운영 회신
  • 최소 집행금액 나스미디어 내부 자료

    광고주 전달용 슬라이드는 "최소 집행 약정 400만원 / 상세 조건은 영업 담당 안내" 톤으로 표기합니다. 크리테오 경유 상세 조건은 내부 영업 안내로 처리합니다.

  • 입찰/과금 방식 나스미디어 내부 자료

    OpenAI 직접 집행은 CPC와 CPM 선택 및 최대 입찰가 조정이 가능합니다. 크리테오 경유는 CPM만 가능하며 입찰 조정은 불가합니다.

  • 수수료/마크업 나스미디어 내부 자료

    OpenAI 직접과 크리테오 경유 모두 마크업 진행이 필요합니다. 크리테오 호스팅사 연동 운영 시 호스팅 fee 5%가 발생합니다.

  • VAT 나스미디어 내부 자료

    한국 사업자등록번호(BRN) 또는 세금ID 제공 시 0%, 미제공 시 10%로 적용됩니다.

  • 전환 추적(트래커) 나스미디어 내부 자료

    노출/클릭 캠페인에서는 필수는 아니지만 강력 권장됩니다. 향후 전환 최적화 캠페인은 전환 추적 설정이 필요합니다.

  • 초기 노출 나스미디어 내부 자료

    한국 런칭 초기에는 전체 광고 자격 유저의 10%에게 노출 후 점진 확대됩니다. 크리테오 경유 캠페인도 동일 적용되며 예외는 없습니다.

  • 인보이스(OpenAI) 나스미디어 내부 자료

    나스미디어가 광고 계정 전체를 통합해 단일 인보이스를 발행하고, 인보이스 내 광고주별 집행금액을 개별 표기합니다. 실제 집행분 기준 익월 첫 영업일 7일 이내 발행됩니다.

  • 인보이스(크리테오) 나스미디어 내부 자료

    기존 크리테오 광고 상품 정산과 동일 방식 예정입니다. 단, 추후 변경될 수 있습니다.
